# ⚡ digiTools Platform

**digiTools** is a **responsive platform app** built with **React**, providing a collection of **digital productivity and utility tools**. This project showcases **frontend skills, state management, interactive components, and responsive UI design**.

---

## 🚀 Features

- A collection of **digital tools** for productivity, data management, and utilities.  
- **Responsive UI** using **Tailwind CSS** and **DaisyUI**.  
- Dynamic **state management** with React Hooks and Context API.  
- Modular and reusable **React components**.  
- **Data persistence** for applicable tools using localStorage.  
- Clean and modern **frontend design** with smooth user experience.
