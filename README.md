\*\*Name of Project:
Digitools-platform-2026.

\*\*A little Description:
Digitools is a user-friendly platform offering a collection of AI-powered and productivity tools for everyday tasks. It focuses on speed, accessibility, and a clean UI to enhance user experience.

\*\* Technology Used:
Tailwind CSS, daisyUI, ReactJS, react-lucide, react-toastify, JSON(for fake data)

\*\*3 Features about project:
--> The project has two tabs, one is for all products, and another is for selected products, named 'cart'.
--> After select, The 'Buy now' button will be converted to 'Add item to the cart!' .One cart component will be appeared and show all selected items in the cart tab.
--> User can add/remove an item in cart component, and the price will be updated in every add and remove.
